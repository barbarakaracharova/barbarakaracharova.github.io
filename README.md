# Сайт Варвары Лузан

Персональный сайт-портфолио для психолога на React, TypeScript, Tailwind, Framer Motion и Vite.

## Локальный запуск

```bash
pnpm install
pnpm dev
```

## Сборка

```bash
pnpm build
```

## GitHub Pages

В проект добавлен workflow `.github/workflows/deploy.yml`. Для публикации включите GitHub Pages в настройках репозитория и выберите источник **GitHub Actions**.
